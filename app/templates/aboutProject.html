About the Project

The Court Data Fetcher & Mini-Dashboard project is a web-based platform designed to simplify access to court case details from the Faridabad District Court. Users can input a case number and retrieve case status, petitioner/respondent names, next hearing date, and other metadata.

Built with Flask and Python, this tool integrates web scraping (via Selenium) to fetch real-time data directly from the official court website. Each query is also logged into a database for record-keeping and analysis. The application features a user-friendly interface and a live dashboard to display search trends.

  • Simple case status lookup by case number
  • Dashboard displaying recent activity
  • Persistent search logging to database
  • Clean and professional UI with modular routing

This project is ideal for law firms, legal researchers, and citizens who need quick access to court case information. Future expansions may include support for additional courts, downloadable case reports, and notification alerts for hearing updates.
